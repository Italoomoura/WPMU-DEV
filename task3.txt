WordPress is a system used to manage websites, virtual stores, news portals and other types of pages, the objective is to manage content in a simple and practical way. WordPress is important because it makes it easy for everyone to make a website simple.
A plugin is code inserted into your WordPress site. Simply put, it is an extension that enhances and improves the functionality of your main site. Plugins are capable of turning blogs into websites of any kind. In a few simple steps a blog can be transformed into virtual stores, magazines, etc. Some plugins completely transform the site, others just make small changes. Even not visible, they help with optimization to improve site performance.
It's like WordPress is a person and the plugin is a vitamin supplement.
